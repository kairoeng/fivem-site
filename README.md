# NorthGrid Store

NorthGrid Store is a modern marketplace for premium FiveM scripts.

It is built for RP communities that need a clean storefront, fast access to products, and simple support workflows.

## Features
- Product catalog with detailed product pages
- Social sign-in (Discord and Google OAuth)
- User dashboard for account and purchases flow
- Built-in ticket system for support
- Admin panel with key store stats

## Tech Stack
- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4

## Quick Start
```bash
npm install
npm run dev
```

Open: `http://localhost:3000`

## Environment
Copy `.env.example` to `.env` and fill your values:
- `APP_URL`
- `JWT_SECRET`
- `DISCORD_CLIENT_ID`, `DISCORD_CLIENT_SECRET`, `DISCORD_REDIRECT_URI`
- `GOOGLE_CLIENT_ID`, `GOOGLE_CLIENT_SECRET`, `GOOGLE_REDIRECT_URI`

## Scripts
```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Notes
- This repository is prepared for public GitHub publishing.
- Sensitive values are removed and replaced with placeholders.
